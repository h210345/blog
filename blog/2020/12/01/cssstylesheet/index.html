CSSStyleSheet

Tue Dec 01 2020
  • CSSOM
  • CSS

CSSStyleSheet接口代表一个 CSS 样式表，并允许检查和编辑样式表中的规则列表。它从父类型 StyleSheet 继承属性和方法

在文档的样式表加载时，一个 CSSStyleSheet 对象由浏览器自动创建并插入至文档的Document.styleSheets列表中。由于样式表列表不能直接修改，我们没有什么有效的手段取手动创建一个新的 CSSStyleSheet 对象不过Constructable Stylesheet Objects很快会来到web平台，而且Google 渲染引擎Blink早已支持 (opens new window)。需要创建新的样式表就直接在文档中插入
